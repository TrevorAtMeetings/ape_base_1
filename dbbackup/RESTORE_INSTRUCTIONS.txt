OFFLINE DATABASE RESTORE INSTRUCTIONS
======================================

To restore this backup to a local PostgreSQL database:

1. PREREQUISITES:
   - PostgreSQL installed locally (version 13 or higher recommended)
   - pg_restore utility available

2. CREATE A NEW DATABASE (if needed):
   createdb ape_pumps_local

3. RESTORE THE BACKUP:
   pg_restore --dbname=ape_pumps_local --no-owner --no-acl --clean --if-exists --verbose ape_pumps_offline_backup_*.dump

   Or with connection string:
   pg_restore --dbname=postgresql://username:password@localhost:5432/ape_pumps_local --no-owner --no-acl --clean --if-exists --verbose ape_pumps_offline_backup_*.dump

4. VERIFY THE RESTORE:
   psql -d ape_pumps_local -c "SELECT COUNT(*) FROM pumps;"
   psql -d ape_pumps_local -c "SELECT COUNT(*) FROM pump_performance_points;"

5. EXPECTED DATA:
   - 386 pumps
   - 869 curves
   - 7,043+ performance points
   - All Brain system calibration data

NOTES:
- The backup is in PostgreSQL custom binary format (compressed)
- It includes all tables, data, indexes, and constraints
- No user/role information is included for portability
- The --clean flag means it will DROP existing objects before recreating
